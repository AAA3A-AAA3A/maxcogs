<h1 align="center">MAXCOGS</h1>

<img src="https://i.imgur.com/9WesmVE.png" alt="Banner" style="max-width:100%;">


To add this: 

`[p]repo add maxcogs https://github.com/maxbooiii/maxcogs`

To install:

`[p]cog install maxcogs <cog name you want>`
